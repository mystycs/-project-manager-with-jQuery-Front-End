## Project-Manager

A Ruby on Rails framework driven app with a jQuery Front End which is designed to manage projects and tasks.

## Running Locally

```  
git clone git@github.com:mystycs/project-manager.git # or clone your own fork
cd project-manager
bundle install
rake db:migrate
```
